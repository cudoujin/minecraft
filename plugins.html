Interaction Visualizer

https://www.spigotmc.org/resources/77050/
Interaction Visualizer為Minecraft中不同的互動加入了動畫。當你在工作台合成物品時，放入的物品會顯示在工作台上方，它還增加了進度條和交易動畫，詳細功能請到以上網站查看。
指令:
/iv toggle itemstand (animation_name) true/false
開關顯示靜態物品，例如合成枱和製圖台上的物品展示。
/iv toggle itemdrop (animation_name) true/false
開關顯示掉落物的動畫，例如掉落物消失時間和交易物品。
/iv toggle hologram (animation_name) true/false
開關顯示浮空文字，例如進度條。

Interactive Chat

https://www.spigotmc.org/resources/75870/
Interactive Chat能讓你在聊天中展示物品或位置等資訊，跟朋友溝通無難度。只要聊天時加上佔位文字，插件就會以對應資訊替代佔位文字。
示範:

佔位文字:
[i] 或 [item]
展示手持物品。
[inv] 或 [inventory]
展示經驗值和身上所有物品。
[ender]
展示自己終界箱裏的物品。
[gametime]
展示遊戲內太陽對應的時間。
[pos]
展示自己的座標和生態域。
[ping]
展示自己的ping。
[/cmd]
展示可點擊的Minecraft指令，例如 [/tell] 會發送可點擊文字，按下去後會自動輸入/tell指令，但不會發送。
&0 至 &f 或 &#RRGGBB 或 [color=#RRGGBB]
設定文字顏色。
&k / &l / &m / &n / &o / &r
設定文字格式，格式化代碼可到這裡查看。

Minecart Speed Plus

https://www.spigotmc.org/resources/69639/
Minecart Speed Plus新增了透過告示牌調整礦車車速的功能，將Minecraft的高速鐵路夢想化為現實。在加速路軌旁或支撐路軌的方塊下面插上告示牌，並在第一行寫上[msp]，在第二行寫上數字，礦車最高車速就會變成原版車速乘以該數字。想建造過山車的玩家也可以用它提供的礦車飛行功能，玩家只需在第二行寫上fly就能讓礦車跳過路軌之間的空隙，相反nofly則取消飛行效果。
注意!必須使用橡木告示牌!

示範:

Spectator Mode

https://www.spigotmc.org/resources/77267/
